# Wandr – Travel App Landing Page

Wandr is a modern, premium SaaS landing page designed for a travel and itinerary planning application. The design focuses on storytelling, user engagement, and high-conversion UI patterns.

## 📌 Features

- 🌍 Premium hero section with strong visual storytelling
- ✨ Dark-themed modern UI design
- 📱 App showcase sections (Discovery, Itinerary, Booking)
- 🧭 Feature highlights with structured layout
- 📊 Social proof and statistics section
- ⭐ Testimonials for credibility
- 🚀 Conversion-focused CTA sections
- 📱 Fully responsive design
